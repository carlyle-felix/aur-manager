# AURMGR

An Arch User Repository manager written in C.

## MANUAL INSTALLATION

```
$ git clone https://github.com/carlyle-felix/aur-manager.git
$ cd aurmgr
$ make
# make install
```

## USAGE

| COMMAND | DESCRIPTION |
| ------- | ----------- |
|`aurmgr -u` | update |
|`aurmgr -i [git clone URL]` | clone and install |
